Exclusao de Dados

Ultima atualizacao: 2026-04-30

O app Protetor de Chamadas (com.intelmib.callblocker) nao possui conta de usuario, login, servidor proprio ou sincronizacao externa. Os dados usados pelo app ficam no dispositivo do usuario.

Como excluir dados no Android

  1. Abra as configuracoes do Android.
  2. Entre em Apps ou Aplicativos.
  3. Selecione Protetor de Chamadas.
  4. Abra Armazenamento.
  5. Toque em Limpar dados ou Apagar dados.

Desinstalacao

Ao desinstalar o app, o Android remove os dados locais associados ao aplicativo, incluindo regras e registros armazenados no espaco privado do app.
