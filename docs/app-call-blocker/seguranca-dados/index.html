Seguranca dos Dados

Ultima atualizacao: 2026-04-30

O app Protetor de Chamadas (com.intelmib.callblocker) processa dados relacionados a chamadas no proprio dispositivo do usuario para aplicar regras de bloqueio configuradas localmente.

Resumo

ItemDeclaracao
Compartilhamento com terceirosO app nao vende nem compartilha dados pessoais com terceiros.
Envio para servidor proprioO app nao possui login, conta, servidor proprio ou sincronizacao externa.
Processamento localNumeros, regras e historico sao usados localmente para bloquear chamadas e exibir registros no app.
ExclusaoO usuario pode remover dados do app pelo proprio aplicativo, quando disponivel, ou pelas configuracoes do Android.

Dados usados localmente

  • Estado do telefone e eventos de chamada.
  • Numero de chamadas recebidas, quando disponibilizado pelo Android.
  • Historico local de chamadas analisadas e bloqueadas.
  • Regras de bloqueio criadas pelo usuario.

Finalidade

Esses dados sao usados para identificar chamadas, comparar com regras configuradas pelo usuario, bloquear chamadas indesejadas e exibir historico local dentro do app.
